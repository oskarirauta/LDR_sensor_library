This is an library for Leaflabs's Maple. Library supports series of low cost LDR sensors. 
LDR sensor is a light dependant resistor, a device that can be used to measure amount of
light.

This is a simple library for light sensing with LDR.
Written by Oskari Rauta.
License: Public Domain

To download: click the DOWNLOADS button in the top right corner, rename the uncompressed folder LDR_sensor_library. Check that the folder contains LDR.cpp and LDR.h. Place the library folder your <maplesketchfolder>/libraries/ folder. You may need to create the libraries subfolder if its your first library. Restart the IDE. Connect LDR another pin to VCC and another to 10K resistor and PIN 18. Connect resistors other end to GND.
Run example and check serial output.
